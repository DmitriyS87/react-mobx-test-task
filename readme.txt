
Дмитрий привет
Вот тестовое задание
Необходмио создать небольшое приложение SPA.
Стек: react + mobX +  typescript + в качестве библиотеки UI использовать Ant Design. Компоненты выполнить в виде классов.

Приложение взаимодействует с rest api https://gorest.co.in/.

В приложении должно быть меню с пунктом Users.
Страница пользователей открывает список пользователей с пагинацией.
Должна быть возможность перейти к списку постов пользователя (Отдельная страница)
При выделении строки с пользователем становится активна кнопка на панели таблицы (перейти к постам)

На странице Posts таблица постов с пагинацией.
Должна быть возможность просмотра комментарев к посту.
При выделении строки с постом снизу должен обновляться список комментарев
в коментарии информаци о пользовтеле и текст
